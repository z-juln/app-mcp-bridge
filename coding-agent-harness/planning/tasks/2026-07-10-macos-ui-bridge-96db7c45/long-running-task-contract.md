# 实现 macOS 通用 UI Bridge 第一轮 - 长程任务合同

## 目标

持续推进到第一轮通用 macOS 服务、HTTP、MCP、Skill 和代表应用证据全部完成。

## 范围

- 范围内：本仓全部第一轮代码、测试、Skill、文档和 Harness 记录。
- 范围外：Windows、产品 UI、正式发布、云视觉和应用专属脚本。
- 冲突风险：共享任务文档和协议模型由当前 coordinator 串行维护。

## 主入口

- 主调用方：Cursor、WorkBuddy 和本地 Agent。
- 必须支持：本地 HTTP、MCP stdio、MCP Streamable HTTP、Skill。
- 不要求：人类产品 UI、远程网络访问。

## 执行授权

- 连续执行：allowed。
- 每轮无需重新询问：yes。
- 审查子代理：本任务不启用，以节省用户额度；需要时必须重新说明收益。
- 必须人工批准：对外发送、删除、购买、系统安全设置、凭据操作、架构主方向改变。

## 必需循环

每个切片执行：实现 → 本地运行 → 测试/冒烟 → 自审 → 修复 → 重测 → 更新进度 → 提交。
